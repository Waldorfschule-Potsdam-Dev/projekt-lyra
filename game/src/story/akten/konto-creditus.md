# 5. TRANSAKTIONSLOGIK

## 5.1 Steuerungsprinzip

Finanzbewegungen erfolgen nicht isoliert, sondern innerhalb eines Regelkreises:

1. Bedarfsermittlung (Atlas / Spiegel)  
2. Prioritätszuweisung (P-System)  
3. Budgetfreigabe (Finanzen)  
4. Projektausführung (Organisation / Umsetzung)  
5. Rückkopplung (Prisma / Analyse)  

---

## 5.2 Charakter der Bewegungen

- überwiegend interne Umbuchungen  
- projektgebundene Allokationen  
- keine linearen Einzelausgaben, sondern systemische Verteilung  
- zeitlich gebündelte Mittelverschiebungen zur Stabilisierung
